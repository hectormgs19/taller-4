# Biblioteca El Progreso

Este proyecto es una aplicación básica hecha en JavaScript para la Biblioteca del barrio El Progreso.

El programa funciona desde la consola del navegador y permite registrar usuarios, registrar libros, prestar libros, devolver libros y consultar información de los libros.

## ¿Qué hace el programa?

La aplicación permite realizar las siguientes acciones:

1. Registrar usuarios con:
   - Nombre
   - Correo electrónico

2. Registrar libros con:
   - Título
   - Autor
   - Estado del libro: disponible o prestado

3. Prestar un libro:
   - El programa valida si el libro existe.
   - El programa valida si el libro está disponible.
   - Si está disponible, cambia su estado a prestado.

4. Devolver un libro:
   - El programa valida si el libro existe.
   - Si el libro estaba prestado, cambia su estado a disponible.

5. Consultar libros:
   - Ver todos los libros.
   - Ver solo los libros disponibles.
   - Buscar libros por autor.
